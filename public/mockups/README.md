# Device Mockups for Live Preview

Thư mục này chứa các hình ảnh mockup thiết bị để hiển thị preview trong admin.

## Files cần có:

- `desktop-mockup.png` - Mockup màn hình desktop
- `tablet-mockup.png` - Mockup tablet 
- `mobile-mockup.png` - Mockup điện thoại

## Kích thước khuyến nghị:

- Desktop: 1200x800px
- Tablet: 800x1000px  
- Mobile: 400x800px

## Nguồn mockup miễn phí:

- https://mockuphone.com/
- https://deviceshots.com/
- https://www.figma.com/community (search "device mockup")
- https://www.freepik.com/free-photos-vectors/mockup

## Lưu ý:

Mockup nên có vùng trong suốt hoặc màu trắng ở giữa để hiển thị nội dung preview.
